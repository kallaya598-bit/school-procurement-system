# 📖 คู่มือการใช้งานอย่างรวดเร็ว

## ✨ อะไรที่เปลี่ยนแปลง

ระบบเอกสารพัสดุได้ถูกอัพเดท ดังนี้:

### 1️⃣ ตราครุฑแสดงที่หน้าบันทึกข้อความ
- 🎖️ ตราครุฑ (ตราราชการ) จะแสดงที่มุมซ้ายบน
- 📄 อยู่บนหน้าบันทึกข้อความทั้งสัก 3 หน้า

### 2️⃣ ตารางสรุปข้อมูลโครงการ
- 📊 แสดง "ดังเอกสารที่แนบมา"
- 📋 แสดงจำนวนโครงการ = 1
- 💰 แสดงชื่อโครงการและจำนวนเงินทั้งสิ้น

## 🚀 ขั้นตอนการใช้งาน

### ขั้นตอน 1: เตรียมเครื่องคอมพิวเตอร์
```bash

```

### ขั้นตอน 2: เรียกใช้ระบบ
```bash
python server.py
```

### ขั้นตอน 3: เปิดเว็บบราวเซอร์
```
http://127.0.0.1:8087
```

### ขั้นตอน 4: กรอกข้อมูล
- ✏️ กรอกข้อมูลทั่วไป (วันที่, เดือน, พ.ศ.)
- ✏️ กรอกชื่อโครงการ
- ✏️ เพิ่มรายการพัสดุ
- ✏️ กรอกชื่อเจ้าหน้าที่ (หากต้องการ)

### ขั้นตอน 5: สร้างไฟล์ Word
- 🖱️ คลิก "สร้างไฟล์ Word" (ปุ่มสีฟ้า)

- 📥 คลิก "ดาวน์โหลดไฟล์ Word"

## 🌐 อัพโหลดไปใช้บนเว็บ (Render.com)

### ขั้นตอน 1: เตรียมบน GitHub
```bash
git init
git add .
git commit -m "School procurement system"
git push origin main
```

### ขั้นตอน 2: เชื่อมต่อ Render
1. ไปที่ https://render.com
2. เข้าสู่ระบบ (หรือลงทะเบียน)
3. คลิก "New+" → "Web Service"
4. เลือก GitHub repository ของคุณ
5. ตั้งค่า:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python server.py`
   - **Environment**: `HOST = 0.0.0.0`

### ขั้นตอน 3: เปิดใช้งาน
- ⏳ รอให้ Render สร้างและเรียกใช้งาน (10-15 นาที)
- ✅ จะได้ URL เช่น `https://your-app.onrender.com`
