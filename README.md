# ระบบเอกสารพัสดุโรงเรียน

ระบบนี้ใช้สำหรับกรอกข้อมูลขอซื้อ/ขอจ้างครั้งเดียว แล้วสร้างไฟล์ Word ชุดเอกสารพัสดุตามแบบฟอร์มโรงเรียน

## วิธีใช้

1. เปิดไฟล์ `เปิดระบบพัสดุ.bat`
2. กรอกข้อมูลโครงการ ผู้รับผิดชอบ รายการพัสดุ และคณะกรรมการ
3. กด `สร้างไฟล์ Word`
4. ดาวน์โหลดเอกสารที่ระบบสร้างไว้ในโฟลเดอร์ `generated`

## การนำไปให้ผู้อื่นใช้

คัดลอกทั้งโฟลเดอร์ `school-procurement-system` ไปยังเครื่องอื่น แล้วเปิดไฟล์ `เปิดระบบพัสดุ.bat`

หากเครื่องอื่นเปิดไม่ได้ ให้ติดตั้ง Python แล้วติดตั้งแพ็กเกจด้วยคำสั่ง `pip install -r requirements.txt`

## หมายเหตุ

ไฟล์แบบฟอร์มต้นฉบับถูกเก็บไว้ที่ `templates/original-request-form.docx` เพื่อใช้อ้างอิงต่อยอดในอนาคต
