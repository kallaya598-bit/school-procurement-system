## 🔧 ฟังก์ชัน add_summary_table() - รายละเอียด

### ลักษณะการทำงาน

```python
def add_summary_table(doc, project_name, total_amount, font_size=14):
```

### พารามิเตอร์ที่รับเข้า

| พารามิเตอร์ | ประเภท | คำอธิบาย |
|-----------|--------|---------|
| `doc` | Document | วัตถุ python-docx ที่ใช้แก้ไขเอกสาร |
| `project_name` | str | ชื่อโครงการที่ดึงมาจาก `data["project"]` |
| `total_amount` | float | จำนวนเงินรวมที่คำนวณจากรายการ |
| `font_size` | int | ขนาดฟอนต์ (ค่าเริ่มต้น: 14) |

### ขั้นตอนการทำงาน

1. **เพิ่มข้อความหัวข้อ**: "ดังเอกสารที่แนบมา" (ตัวหนา, ขนาด 14)
2. **สร้างตาราง**: 3 แถว × 2 คอลัมน์
3. **แถวที่ 1 (หัวตาราง)**: 
   - ซ้าย: "รายการ" (สีพื้นหลัง D9EAF7, จัดกลาง)
   - ขวา: "ข้อมูล" (สีพื้นหลัง D9EAF7, จัดกลาง)
4. **แถวที่ 2**:
   - ซ้าย: "จำนวนโครงการ" (จัดกลาง)
   - ขวา: "1" (จัดกลาง)
5. **แถวที่ 3**:
   - รวมทั้ง 2 คอลัมน์
   - เนื้อหา:
     ```
     โครงการ: [ชื่อโครงการ]
     จำนวนเงินทั้งสิ้น: [จำนวน] บาท
     ```

## 🖼️ การจัดการรูปภาพตราครุฑ

### ที่มาของการเปลี่ยนแปลง

```python
# เดิม (ไฟล์หายไป):
GARUDA_PATH = BASE_DIR / "templates" / "image2.png"

# ใหม่ (ใช้ไฟล์ที่มี):
GARUDA_PATH = BASE_DIR / "images.jpg"
```

### วิธีการตรวจสอบความมีตัวตน

```python
# ในฟังก์ชัน add_memo_header():
if garuda_position and GARUDA_PATH.exists():
    p_img = doc.add_paragraph()
    p_img.alignment = WD_ALIGN_PARAGRAPH.LEFT if garuda_position == "left" else WD_ALIGN_PARAGRAPH.CENTER
    p_img.paragraph_format.space_before = Pt(0)
    p_img.paragraph_format.space_after = Pt(0)
    p_img.paragraph_format.line_spacing = 1.0
    p_img.add_run().add_picture(str(GARUDA_PATH), height=Cm(2.25))
```

**การทำงาน**:
1. ตรวจสอบ `garuda_position` มีค่า และ `GARUDA_PATH` มีไฟล์
2. ถ้ามี ให้สร้างย่อหน้าใหม่
3. ตั้งค่าการจัดแนวตามตำแหน่ง (ซ้าย/กลาง)
4. เพิ่มรูปภาพด้วยขนาดความสูง 2.25 เซนติเมตร

## ⚙️ การตั้งค่าและปรับแต่ง

### หากต้องการเปลี่ยนสีพื้นหลังตาราง

ในฟังก์ชัน `add_summary_table()` ค้นหา:
```python
set_cell_shading(header_left, "D9EAF7")  # สีฟ้าอ่อน
set_cell_shading(header_right, "D9EAF7")
```

เปลี่ยน hex color code (เลือกจากไพลต์ Excel):
- `D9EAF7` = ฟ้าอ่อน (ปัจจุบัน)
- `E2EFDA` = เขียวอ่อน
- `FCE4D6` = ส้มอ่อน

### หากต้องการปรับขนาดตราครุฑ

ในฟังก์ชัน `add_memo_header()`:
```python
p_img.add_run().add_picture(str(GARUDA_PATH), height=Cm(2.25))  # เปลี่ยน 2.25 เป็นค่าใหม่
```

### หากต้องการเพิ่มเนื้อหาในตารางสรุป

แก้ไขฟังก์ชัน `add_summary_table()` เพิ่มแถวใหม่:
```python
# หลังแถวที่ 2:
cell_paragraph(table.cell(2, 0), "รายการใหม่", size=font_size, align=WD_ALIGN_PARAGRAPH.CENTER)
cell_paragraph(table.cell(2, 1), "ค่าใหม่", size=font_size, align=WD_ALIGN_PARAGRAPH.CENTER)
```
